# Windows桌面系统模拟器

一个基于HTML、CSS和JavaScript构建的Windows桌面系统模拟器，具有完整的桌面环境和应用程序功能。

## 功能特点

### 1. 桌面环境
- 完整的Windows风格桌面界面
- 可自定义桌面背景和图标
- 支持动态壁纸和幻灯片放映
- 右键菜单功能

### 2. 应用程序
- **文件资源管理器**：支持文件上传、下载、创建文件夹和文件管理
- **记事本**：支持文本编辑和自动保存
- **计算器**：基本计算功能
- **浏览器**：访问网页
- **媒体播放器**：播放本地和在线媒体文件
- **设置**：系统设置和个性化

### 3. 本地存储功能
- 文件管理器中的文件永久保存在本地
- 记事本内容自动保存
- 本地图片库支持上传和管理图片
- 自定义网站图标保存

### 4. 回收站功能
- 支持删除文件、图片和网站
- 可恢复删除的项目
- 可永久删除项目
- 清空回收站功能

## 使用说明

### 文件管理器
1. 点击"此电脑"或"文件资源管理器"打开文件管理器
2. 在左侧导航栏选择文件夹
3. 点击"上传文件"按钮上传文件
4. 点击"新建文件夹"创建新文件夹
5. 右键点击文件可以打开或删除文件

### 本地图片库
1. 打开设置
2. 选择"本地图片"选项卡
3. 点击"选择图片"并选择要上传的图片
4. 点击"上传"按钮保存图片
5. 右键点击图片可以删除（会移到回收站）

### 记事本
1. 点击"记事本"图标打开
2. 输入文本内容（会自动保存）
3. 关闭后再次打开会恢复之前的内容

### 回收站
1. 打开设置
2. 选择"回收站"选项卡
3. 点击"恢复"按钮恢复项目
4. 点击"永久删除"按钮彻底删除项目
5. 点击"清空回收站"删除所有项目

## 测试功能

打开 `test.html` 文件可以测试所有本地存储功能是否正常工作。

## 注意事项

1. 所有数据都存储在浏览器的本地存储中，清除浏览器数据会导致数据丢失。
2. 本地存储有大小限制，不建议存储过大的文件。
3. 某些浏览器可能限制本地存储的访问，请确保浏览器设置允许本地存储。

## 技术实现

- HTML5 + CSS3 + JavaScript
- 使用localStorage进行本地数据存储
- 使用FileReader API处理文件上传
- 使用DataURL存储图片和小文件
